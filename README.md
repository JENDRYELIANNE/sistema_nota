⚪Manejo de excepciones de un sistema de nota
------------------------------------------------------------
Descripción

este sistema trata de la recoleccion de los datos del estudiantes (el nombre y la nota). 
que mediante estos datos se visualizara si aprobo o reprobo, dependiendo la nota puesta, si pones letra en vez de numeros
te manda un mensaje, al igual que si la nota pasa de 100, se le mandara un mensaje.

⚪Tecnologías Utilizadas
------------------------------------------------------------
•Lenguaje: C#

•Framework: Aplicacion de consola

•Entorno de Desarrollo (IDE): Visual Studio

•Base de Datos: No aplica

⚪Funcionalidades del Sistema
------------------------------------------------------------
•Captura el nombre del estudiante
•Captura la nota final del estudiante
•Resultado final claro (Aprobado / Reprobador)

⚪Uso o Ejecución
------------------------------------------------------------
1-Abrir el proyecto en Visual Studio
2-Ejecutar el sistema
3-Ingresar el nombre del estudiante y la nota
5-El sistema pondra si aprobo o no

Lo que entendí: 
-----------------------------------------------------------
El manejo de excepciones permite controlar errores sin que el programa se cierre inesperadamente. En esta práctica se utilizó Try-Catch-Finally para validar la entrada de una nota. El bloque try ejecuta el código principal, catch detecta y maneja errores como ingresar letras en lugar de números, y finally muestra un mensaje al finalizar el proceso. Esto hace que la aplicación sea más segura, estable y fácil de usar.
